Discord-based haiku session bot

Kukai Bot即開設。即句会。

Discordさえあれば、句会をしたいサーバにbotを導入するだけ。申し込み不要ですぐに句会を開始できます。

資料を見る 問い合わせ

機能

句会運営に必要な作業をまとめて管理

01

参加受付

俳号付きのエントリー、承認制、締切後エントリー、参加状況確認に対応します。

02

投句・選句

GUI入力と一括入力に対応し、投句数や選句数の条件を句会ごとに管理します。

03

結果公開

点数順、番号順、作者別の表示を切り替え、総評や作者コメントも扱えます。

04

通知

締切前通知、未達者通知、管理者向け警告、ボイス句会通知を設定できます。

05

PDF出力

投句一覧PDFと結果PDFを作成し、作者公開設定に応じて出力内容を制御します。

06

運用管理

Docker Compose と PostgreSQL を前提に、本番運用、バックアップ、復旧手順を整備しています。

基本フロー

手動、半自動、全自動の進行に対応
